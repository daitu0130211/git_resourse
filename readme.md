### git 常用指令学习

### 配置用户名和邮箱
git config --global user.name "your name"
git config --global user.email "your email"

### 初始化仓库
git init

### 添加文件(暂存登记到git仓库)
git add .  
git add <file> 
git add <file1> <file2> ...

### 查看git仓库状态
git status

### 提交文件(提交到仓库)
git commit -m "提交说明"
